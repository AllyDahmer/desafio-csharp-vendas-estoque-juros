# Desafio - Cálculo de Comissão, Estoque e Juros em C#

Este projeto foi desenvolvido como parte de um processo seletivo para desenvolvedor.

## Funcionalidades

1. **Comissão de vendedores**
   - < R$100 → sem comissão
   - R$100 até R$499,99 → 1%
   - >= R$500 → 5%
   - Resultado agrupado por vendedor

2. **Controle de estoque**
   - Entrada e saída de produtos
   - Geração de identificador único (GUID)
   - Retorno do estoque final

3. **Cálculo de juros**
   - Multa de 2,5% ao dia de atraso

## Ferramentas

- C# (.NET 6+)
- Console Application
- Git e GitHub
